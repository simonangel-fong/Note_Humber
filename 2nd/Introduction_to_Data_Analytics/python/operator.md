## Arithmetic Operator

```py

x = 5
y = 6.6

print(x+y)  # addition
print(x-y)  # substraction
print(x*y)  # multiplication
print(x/y)  # division
print(x**2)  # Exponentiating
print(y%x)  # Modulus

```
