## Syntax

- `Backslash`: lets you continue your block of code on the next line.

```py

num_apple = 5
num_banana = 4

# use backslash indicates the block continues on the next line.
total_cost = num_apple*1.25 + \
    num_banana*.89

print(total_cost)



```
